# Fusion Marketing Management

Premium 3D digital agency website rebuilt with React, Vite, TypeScript, Tailwind CSS, Framer Motion, and React Three Fiber.

## Quick start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — local development server
- `npm run build` — production build
- `npm run preview` — preview production build

## Stack

- React 19 + Vite + TypeScript
- Tailwind CSS v4 + shadcn-style `components/ui`
- Framer Motion
- React Three Fiber / Drei (hero only)
- React Router
- Lucide React

## Notes

- Static HTML backup lives in `_static_backup/`
- Contact form uses a demo handler; set `VITE_FORM_ENDPOINT` in `.env` for real submissions
- Phone / WhatsApp numbers and case metrics are placeholders in `src/data/`
