
# Prompt 1: HTML/CSS/JavaScript Design

Design a complete 5-page digital marketing agency website using HTML, CSS, and JavaScript. The website should look modern, trustworthy, premium, and fully responsive.

Use the attached sample file as inspiration (FusionHero.jsx / FusionHero.css / fusion-hero-preview.html). Take inspiration from its layout, typography, color system (paper/ink base with a cobalt-to-ochre fusion accent), and overall user experience while creating a cohesive full site.

Business Details:
- Business name: Fusion Marketing Management
- What we do: A Dubai-based digital marketing agency helping business owners, entrepreneurs, and corporates grow through Google Ads and Facebook Ads campaigns, AI automation and AI-powered solutions, social media marketing, e-commerce advertising, lead generation, and website development — turning digital strategy into measurable sales and inquiries.
- Target audience: Business Owners, Entrepreneurs, Corporate
- Main goal: Generate leads, increase sales, encourage inquiries, promote AI automation and AI solutions
- SEO keywords: Digital marketing, Google ads, Facebook ads, Ai automations expert

Build These 5 Pages:
- Home: Hero with primary CTA, services overview (6 services), why choose us, results/stats, testimonials, final CTA
- About: Company story, mission, the "Fusion" philosophy (bringing disciplines together to create stronger results), Dubai/UAE market focus, team
- Services: Detailed breakdown of AI Automation, Social Media Marketing, E-commerce Ads, Website Development, Lead Generation, and Digital Content Creation
- Case Studies: Results-driven case studies or portfolio highlights, client logos, key metrics (leads generated, ROAS, growth)
- Contact: Contact form, business details, Dubai address, map placeholder, business hours, and social links

Requirements:
- Sticky header with logo, navigation, CTA button, and mobile menu
- Footer with quick links, contact details, and social media
- High-quality business/marketing imagery using Unsplash placeholders (teams, dashboards, ad campaigns, offices) — no automotive imagery
- Modern UI with subtle animations and hover effects
- Fully mobile responsive
- SEO-ready with semantic HTML, proper headings, meta tags, and descriptive alt text

# Prompt 2: WordPress Conversion

Convert this design into a WordPress website using Elementor. Use native Elementor widgets wherever possible and build layouts with Containers instead of Inner Sections. The Atomic Editor is disabled, so do not use Atomic Elements. The Novamira plugin is installed to help you understand and structure Elementor elements more effectively.

Create a new WordPress menu for all pages created as part of this design and set the appropriate page as the homepage.

For the header and footer, use XPRO to build custom templates with Elementor. Do not use Elementor Pro widgets. Save both the header and footer templates to the XPRO library so they can be easily customized later.

Use WPForms for all contact and inquiry forms on the site. Use novamira to help you.

Once the conversion is complete, publish the website to the connected WordPress site.
